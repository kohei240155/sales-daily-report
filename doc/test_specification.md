## 概要

### テスト目的

営業日報システムが要件定義に基づいて正しく動作することを確認する

### テスト対象

- バージョン：1.0
- 対象範囲：全機能

### テストレベル

1. 単体テスト（Unit Test）
2. 結合テスト（Integration Test）
3. システムテスト（System Test）
4. 非機能テスト（Non-functional Test）

### テスト実施スケジュール

| テストフェーズ | 予定期間   | 担当             |
| -------------- | ---------- | ---------------- |
| 単体テスト     | 開発期間中 | 開発者           |
| 結合テスト     | 2週間      | 開発者・QA       |
| システムテスト | 1週間      | QA・ユーザー代表 |
| 非機能テスト   | 1週間      | QA               |
